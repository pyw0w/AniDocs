# Документация AniCore

Добро пожаловать в документацию для разработчиков AniCore!

## Содержание

### [Разработка плагинов](PLUGIN_DEVELOPMENT.md)
Полное руководство по созданию плагинов для AniCore, включая:
- Создание плагина как отдельного проекта
- Использование API AniCore
- Логирование в плагинах
- Примеры кода

### [Система логирования](LOGGER.md)
Документация по системе логирования:
- Использование в основном коде
- Использование в плагинах
- Настройка уровня логирования
- Формат логов

### [Интеграция с systemd](SYSTEMD.md)
Руководство по интеграции AniCore с systemd:
- Установка и настройка
- Управление сервисом
- Настройка service файла
- Требования

## Для кого эта документация

Эта документация предназначена для:
- **Разработчиков плагинов** - создание собственных плагинов для AniCore
- **Системных администраторов** - настройка и развертывание AniCore
- **Контрибьюторов** - понимание архитектуры и API проекта

## Быстрый старт

1. Хотите создать плагин? Начните с [PLUGIN_DEVELOPMENT.md](PLUGIN_DEVELOPMENT.md)
2. Нужно настроить логирование? См. [LOGGER.md](LOGGER.md)
3. Развертываете на Linux? См. [SYSTEMD.md](SYSTEMD.md)
